![image](https://github.com/toodemhard/astr/assets/100080774/7ac07382-7ae6-4d74-bb84-74bfc9286009)

## Install
### Clone repo
```
git clone https://github.com/toodemhard/astr
cd astr
```
### Prerequisites

Requires CMake and SDL2 (needs to be listed on PATH).

### Build and Run
```
mkdir build
cmake -S . -B build
cmake --build build
./build/astr
```
